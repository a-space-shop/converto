Frequently asked questions

What image formats are supported?
We support JPG, JPEG, PNG and WEBP image formats. If you have a different format try converting it to JPG first.
Will the image quality be preserved?
Yes, we preserve the original image quality when converting to PDF. The resulting PDF will display your image at full resolution.
Can I convert multiple images to one PDF?
Currently each conversion produces one PDF per image. To combine multiple images into one PDF, convert each image then use our Merge PDF tool.
Is there a file size limit?
The maximum file size is 25MB per image. High resolution photos are usually well under this limit.
